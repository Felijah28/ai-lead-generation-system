 ai-lead-generation-system
 
AI-powered lead generation and qualification system built with Voiceflow, n8n, Zapier and Google Sheets


 Overview
A fully automated system that handles the entire lead journey —
from first website visit to booked appointment — without human intervention.

 Tools Used
- Voiceflow (AI Chat Agent)
- n8n (Automation Workflow)
- Zapier (Email Campaign)
- Google Sheets (CRM)
- Google Calendar (Appointment Booking)
- OpenAI (Lead Scoring)
